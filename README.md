# Crab API

## Setup

Create a `.env` file with the following variables:

```sh
PORT=XXXX
DB_URL=XXXXXXXX
```

Run `run-db-setup.sh` to create all required database tables.

## Installation

`npm install -D`

## Development

`npm run dev`
